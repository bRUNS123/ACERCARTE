# ACERCARTE

**Espacio colectivo de arte y dirección**

Una plataforma digital para mostrar, difundir y exhibir arte emergente y contemporáneo. ACERCARTE nace como una página web que conecta artistas, curadores y público en un espacio de encuentro virtual.

---

## 🎯 Visión

Ser el puente entre el arte y las personas — democratizar el acceso a la creación artística, dar visibilidad a talentos emergentes y construir comunidad en torno al arte.

## 🧭 Dirección

- **Exhibir**: Galería virtual con exposiciones curadas
- **Difundir**: Plataforma de difusión para artistas
- **Conectar**: Espacio de encuentro entre creadores y público
- **Colectivo**: Construido y nutrido por la comunidad

## 🚀 Fase actual

Página web estática como punto de partida. Próximas fases: registro de artistas, galería interactiva, tienda, eventos.
